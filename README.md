# 🔍 CodeSherlock

> AI-powered GitHub Repository Intelligence Platform & Developer Assistant

CodeSherlock helps developers quickly understand unfamiliar GitHub repositories using AI. Simply paste a GitHub repository URL to receive an AI-generated summary, detected technology stack, repository metadata, and conversational insights.

---

## ✨ Features

- 🤖 AI-generated repository summary
- 📦 GitHub repository metadata extraction
- ⚙️ Automatic technology stack detection
- 💬 Ask AI questions about any repository
- 🎨 Modern responsive React dashboard
- 🚀 FastAPI backend with modular architecture

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Kanak-Arya-14/CodeSherlock.git
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
